Pensez à remplir pass.php avec vos codes de connexion et d'exécuter sql.txt.

(Edit en 2017)
Le code source n'a pas changé. Mais à l'époque github n'était pas si bien implémenté, et comme j'ai encore une personne par an qui déboule pour me demander le code source, je le publie sur cette plate-forme.
N'hésitez pas à faire des pull requests !
Dans le répertoire screens/, il y a 2300 captures d'écrans de meurtres de l'époque. C'est pour le souvenir. Vous pouvez les supprimer avant de le mettre en ligne.
